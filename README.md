# Python Number Guessing Game

A terminal-based number guessing game built using Python.

The program generates a random number between 1 and 1000, and the player has to guess the correct number using the hints provided by the game.

## Features

- Random number generation
- Unlimited attempts
- Attempt counter
- Higher/lower hints
- Input validation
- Exception handling for invalid inputs

## Concepts Used

- Loops
- Conditional statements
- Exception handling (`try` / `except`)
- Python `random` module
- Variables and counters

## Future Improvements

- Difficulty levels
- GUI version using Tkinter
- Replay option
